Align-items

La propiedad align-items en CSS se utiliza dentro de un contenedor flexible ( display:flex ) para alinear los elementos secundarios ( hijos ) a lo largo del eje secundario ( o transversal ) del contenedor. Este eje perpendicular al eje principal, que es definido por la propiedad flex-direction ( ya sea en fila o en columna )
Los valores que puede tomar son los siguientes:

  1. flex-start:Los elementos secundarios se alinean al inicio del contenedor flexible en el eje transversal.
  2. flex-end:Los elementos secundarios se alinean al final del contenedor flexible en el eje transversal
  3. center:Los elementos secundarios se alinean en el centro del contenedor flexible del eje transversal.
  4. base-line:Los elementos secundarios se alinean en su linea base común dentro del contenedor flexible
  5. strech:Los elementos secundarios se estiran para ocupar todo el espacio disponible en el eje transversal del contenedor.

Propiedades que se aplican a los items hijos

propiedad de items hijos: order

La propiedad order en CSS Flexbox se utiliza para cambiar el orden de los elementos secundarios(hijos) dentro de un contenedor flexible sin cambiar el orden en el código HTML. Esta propiedad toma un valor numerico que determina el orden en el que se renderizaran los elementos dentro del contenedor flexible.
Por defecto todos los elementos secundarios tienen un valor de orden 0. Al asignar un valor numérico diferente a la propiedad order, se puede especificar un nuevo orden de visualizacion para ese elemento. Los elementos con un valor de orden más bajo se muestran antes que los elementos con un valor de ordenar más alto.
Por ejemplo, si tienes tres elementos secundarios dentro de un contenedor flexible y quieres que el tercero se muestre primero, puedes establecer su propiedad ordenar en -1, mientras que los otros elementos pueden permanecer con el valor predeterminado de 0.
En resumen, la propiedad order de Flexbox es una manera conveniente de cambiar el orden visual de los elementod secundarios dentro de un contenedor flexible sin alterar su orden en el codigo HTML.
